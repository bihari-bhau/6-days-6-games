# 🐍 Snake — Day 1 of 6

> Part of the **6 Days · 6 Games** series — one mini game a day, deployable on GitHub Pages.

## Play
Open `index.html` in any browser. No dependencies, no build step.

## Controls
| Key | Action |
|-----|--------|
| `↑ ↓ ← →` or `WASD` | Move |
| `P` | Pause / Resume |
| Swipe | Mobile touch |

## Features
- Smooth canvas rendering with pixelated retro aesthetic
- Leveling system — speed increases every 100 points
- Special golden food (limited time, 3× score)
- Persistent best score via `localStorage`
- Mobile D-pad + swipe support
- Scanline CRT visual effect

## Deploy to GitHub Pages
1. Push this folder (or just `index.html`) to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Done — live at `https://<username>.github.io/<repo>/`
